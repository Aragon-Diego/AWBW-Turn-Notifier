# AWBW Turn Notifier

## How to install using the source code

1. Download the source code
2. Open Google Chrome
3. Go to extensions
4. Click On Manage Extensions
5. Turn On Developer Option
6. Click on load unpdacked
7. Select AWBW-Turn-Notifier folder
8. Enjoy :)
